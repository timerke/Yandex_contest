# Кроссворд
После рефакторинга сломались CSS-стили в кроссворде. Необходимо восстановить стили в соответствии с первоначальным дизайном, но сделать это нужно с минимальным количетсвом исправлений.
## Примечания
Текущие CSS-стили: solution.css. После исправлений, нужно предоставить обновленный CSS-файл. Данный файл будет подключен как solution.css к HTML-странице.
Ваше решение будет тестироваться в браузере Google Chrome 69. Семейство и другие параметры шрифтов изменять не надо. При этом локально у вас может не совпадать шрифт с ожидаемым состоянием, т.к. скриншоты сделаны в Ubuntu. Рекомендуем использовать плагины для pixel-perfect верстки, например, PerfectPixel.
